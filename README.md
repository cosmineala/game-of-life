# game-of-life

### requierments:
npm or yarn

`npm install --global yarn`

<hr>
## Create
1) `yarn create react-app --template typescript <project-name>`
2) `cd <project-name>`

### Used addons:

#### Sass:
`yarn add sass`

<hr>
## Restore:
`yarn install` 

<hr>
## Commands

### Run
`yarn start`

### Build
`yarn build`

<hr>
## Netlify config

Base directory: `gameoflife-react`
Build command: `CI= yarn build`
Publish directory: `gameoflife-react/build`
